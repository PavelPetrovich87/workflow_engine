
# Quickstart: Node UI Verification

## Prerequisites
- Node.js installed
- Project dependencies installed (`npm install`)

## Running the UI

1. **Start the Development Server**:
   ```bash
   npm run dev
   ```

2. **Open the Application**:
   Navigate to the URL provided by Vite (typically `http://localhost:5173`).

3. **Verify Node UI**:
   - Observe the workflow graph.
   - **Click** on any node (e.g., "Trigger", "Calc Random").
   - **Verify**: A modal overlay appears displaying the node's Label and Type.
   - **Action**: Click the "Close" button on the modal.
   - **Verify**: The modal disappears.
